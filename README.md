# CHK Ceylon Tours - Sri Lanka Tourism Website

A premium, eye-catching tourism website built with **React**.

## Features
- **Modern Design**: Glassmorphism, premium typography, and cinematic imagery.
- **Animations**: Smooth fade-in reveals on scroll.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Hero Section**: Breathtaking Sigiriya Rock introduction.
- **Destinations**: Highlighting Ella, Mirissa, Kandy, and Sigiriya.
- **Booking CTA**: Easy-to-use call to action for tour bookings.

## How to Run

### 1. Direct Preview (Recommended for XAMPP)
Simply open the `index.html` file in your browser. This version uses React via CDN and works immediately without any installation.

### 2. React Development Environment
If you prefer a full development setup:
1. Ensure you have [Node.js](https://nodejs.org/) installed.
2. Run `npm install`.
3. Run `npm run dev`.

## Technologies Used
- **React 18**: Frontend UI library.
- **Vanilla CSS**: Custom styling with glassmorphism and modern layouts.
- **Intersection Observer**: For scroll animations.
- **Google Fonts**: 'Outfit' and 'Playfair Display'.
- **FontAwesome**: For beautiful icons.
